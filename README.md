# 💼 Подработки — учёт смен и расчёт зарплаты

Простое PWA-приложение для тех, кто подрабатывает сверх основного графика: журнал смен,
отгулов и отпусков плюс автоматический расчёт зарплаты по реальным правилам — с
переработками, повышенной оплатой в выходные и праздники и отпускными по среднему заработку.

Работает **полностью офлайн**, устанавливается на телефон как обычное приложение, а все
данные хранятся только на вашем устройстве — без регистрации и без сервера.

## ✨ Возможности

### 📅 Журнал смен
- Быстрое добавление подработки в один тап: часы вручную или кнопки-пресеты «3.5 ч» / «8 ч».
- Отгулы (часы, вычитаемые из оклада) и отпуск (по диапазону дат) — тоже в пару тапов.
- Календарь месяца с цветными метками по типу дня (подработка / отгул / отпуск / праздник).
- Карточка «Итого за месяц» с разбивкой: база, ставка за час, доплата за подработки, вычет за отгулы.

### 💰 Расчёт зарплаты
- Считает часовую ставку от оклада и нормы часов, автоматически учитывая рабочие дни месяца.
- Переработка в будни — на выбор один из двух режимов:

  | Часы в будний день | ×1.5 / ×2.0 (по ТК РФ) | ×1.5 на все часы |
  |---|---|---|
  | первые 2 часа | ×1.5 | ×1.5 |
  | остальные часы | ×2.0 | ×1.5 |

- Суббота — ×1.5, воскресенье и праздники — ×2.0.
- **Отпускные** считаются отдельно от оклада: по среднедневному заработку за последние 12
  месяцев (или по значению, заданному вручную), за все календарные дни отпуска, кроме
  попавших на праздники.

### 🎉 Праздники
- Отмечайте праздничные и переносные дни вручную кликом по календарю.
- Или подтяните встроенный производственный календарь РФ одной кнопкой — данные
  применяются мгновенно, без интернета.

### 🗂 Архив
- Просмотр итогов по зарплате за любой месяц — прошедший или будущий — с полным списком
  записей за этот период.

### 📦 Резервная копия
- Экспорт и импорт всех данных (записи, праздники, настройки) в один JSON-файл.
- Ненавязчивое напоминание сделать бэкап, если давно не экспортировали данные.
- Приложение просит браузер не удалять данные при нехватке места на устройстве.

### 📴 Офлайн и PWA
- Service Worker кэширует приложение — оно открывается и работает без интернета.
- Устанавливается на главный экран телефона и выглядит как обычное приложение, без адресной строки браузера.

## 📲 Установка как приложение

Сначала откройте адрес приложения в браузере на телефоне, а затем:

### Android (Chrome / Edge)
1. Откройте страницу приложения.
2. Нажмите на меню **⋮** в правом верхнем углу браузера.
3. Выберите **«Добавить на главный экран»** (или **«Установить приложение»**, если браузер предложит сам).
4. Подтвердите — иконка появится на рабочем столе, приложение будет открываться отдельным окном, без адресной строки.

### iPhone / iPad (Safari)
1. Откройте страницу приложения **именно в Safari** (в других браузерах на iOS кнопки «На экран Домой» для PWA нет).
2. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх) в нижней панели.
3. Выберите **«На экран Домой»**.
4. Нажмите **«Добавить»** — иконка появится на рабочем столе.

После установки приложение работает офлайн и обновляется автоматически при следующем запуске с интернетом — вручную ничего чистить не нужно.
